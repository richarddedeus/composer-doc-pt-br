Comunidade¶

Já existem muitas pessoas usando o Composer e muita delas estão contribuindo.

Contribuindo¶

Se você deseja contribuir com o Composer, leia os documentos README e CONTRIBUTING.

As diretrizes mais importantes são descritas a seguir:

Todas as contribuições de código - incluindo aquelas de pessoas com acesso para fazer commit - devem passar por uma pull request e ser aprovadas por um desenvolvedor principal antes de serem adicionadas a um merge. Isso é para garantir a revisão adequada de todo o código.

Faça o fork do projeto, crie um feature branch e nos envie uma pull request.

Para garantir uma base de código consistente, verifique se o código segue os Padrões de Codificação PSR-2.

IRC / Lista de Discussão¶

Listas de discussão para suporte ao usuário e desenvolvimento.

Canais IRC estão disponíveis no irc.freenode.org: #composer para usuários e #composer-dev para desenvolvimento.

O Stack Overflow tem uma coleção crescente de perguntas relacionadas ao Composer.
